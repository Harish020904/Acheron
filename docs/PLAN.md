# Project Acheron — Showcase Plan

## Objective

Build a deterministic urban simulation showcase that is easy to explain in a university demo.

## Keep

- ECS
- memory allocators
- job scheduler
- simplified renderer
- traffic simulation
- economy simulation
- progression systems
- simplified profiler
- deterministic simulation loop

## Current focus

1. Keep the runtime small and legible.
2. Keep simulation deterministic.
3. Keep generated files inside `build/`.
4. Avoid adding systems that do not strengthen the demo.
